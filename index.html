ระบบแผนผังครอบครัว (Family Tree Web App)

เส้นเชื่อมวาดด้วย JS/SVG วัดตำแหน่งการ์ดจริง ไม่พึ่งเปอร์เซ็นต์ CSS — ตรงเสมอแม้มีคู่สมรสหรือแต่งงานหลายครั้ง

วิธีดูผัง: เส้นดิ่งจากด้านบนจะชี้ตรงลงมาที่ "ลูกสายเลือดตรง (ขอบทึบ)" เสมอ ส่วน "เขย/สะใภ้ (ขอบประ)" จะแยกอยู่ด้านข้างและเชื่อมกันด้วยเส้นสมรสมีหัวใจดวงเล็กๆ คั่นครับ

กำลังประกอบโครงสร้างและจัดวางตําแหน่งเส้นเชื่อมโยง...
